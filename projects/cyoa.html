CYOA Project

World War 3

A WW3 story as a paratrooper where you have to choose wisely to live longer. eflection: For my reflection, a takeaway I was learning was how important it was to plan first before I started coding: the more ideas I had off the bat the more flexibility I had instead of just coding off the back of my head and unsure whether to change an ending or keep it. Mapping the story out first helped with knowing all the possible outcomes good or bad so I was way more organized and could start the project more easily. I also remembered to ask for help or test a lot first to make sure I make no mistakes in my code or there could be an error and ruin the file. A major challenge too for me was first using the CS50 terminal correctly. I was confused on what commands to do first for my work to save, move to my partners side, or what to do in general. But I figured it out soon by asking my teacher and get me just a little better so I know what to do in the future if I run into that problem again. One more challenge was structuring my story to actually make sense, if it didn't make sense then the story wouldn't be taken that seriously, it would have a lot of mistakes and lots of fixing required. If I ever did this project again I would most likely expand the story more with extra alternative endings instead of just 2 good endings, a bit more interactive on user side, and organizing my code so it made more sense. Overall, this project let me learn a bit on how to debug, and learn tools like the CS50 terminal.

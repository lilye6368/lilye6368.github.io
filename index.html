Health & Wellness Website
10th Grade

Health and Wellness Website. The website showcases content of different types of future technology, traditional technology, and advancements that could happen in some time due to the evolution of medicine and research. It is a basic but helpful website with not too much reading, and helps people know more about the current advancements and what's happening in the medical field. I didn't have that many challenges, but I just had to debug the code for the cards a lot and just figure out which animations I wanted to use so it's a little easier to read the text. Two takeaways are that technology can be a very good tool to track and improve personal wellness and that building this project can prove that clean code can simplify complex health descriptions for people.
